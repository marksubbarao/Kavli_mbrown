All Asteroids

A massive system of all known asteroids in our solar system accoring to The Asteroid Orbital Elements Database. (March 2012)

The ASTORB database is managed by Edward Bowell and consists of 588132 orbits.

The module will appaear as "All Asteroids" in the object tree, under Solar System.
